# Portfolio Admin Dashboard

This is a **Portfolio Admin Dashboard** built with [Next.js](https://nextjs.org) using the App Router architecture. It allows dynamic management of portfolio projects, skills, resume, profile image, and contact messages through structured CRUD functionality.

---

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the dashboard by modifying files inside the `app` directory. The page auto-updates as you edit the files.

This project uses modern Next.js features including App Router and TypeScript support.

---

## Features

- Add, update, and delete projects  
- Manage skills and portfolio content  
- Resume upload and preview  
- Profile image management  
- View and delete contact messages  
- Structured component-based dashboard layout
